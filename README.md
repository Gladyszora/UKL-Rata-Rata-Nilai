Menghitung Nilai Rata-Rata Ujian
​Fungsi Program: Untuk membantu guru merekap nilai ujian, termasuk menghitung nilai rata-rata dari keseluruhan nilai.
​Cara Kerja: Program meminta jumlah siswa (N). Menggunakan perulangan, program menerima N nilai ujian satu per satu, menyimpannya dalam array, dan menjumlahkan semua nilai. Nilai rata-rata dihitung dengan membagi Total Nilai dengan Jumlah Siswa (N).
